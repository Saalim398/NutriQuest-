# NutriQuest

NutriQuest is an educational JavaFX game that makes learning about nutrition fun and interactive. Players improve their nutritional knowledge through engaging gameplay, challenges, and healthy food choices.

## Features

* Interactive nutrition based gameplay
* User friendly JavaFX interface
* Educational and entertaining experience
* Score tracking system
* Healthy food awareness
* Lightweight desktop application

---

## Tech Stack

* Java 21
* JavaFX
* Maven
* CSS (for styling)

---

# Running the Game

## Method 1: Run the JAR File (Recommended)

### Requirements

* Java 21 or later installed.

Check:

```bash
java --version
```

### Download

1. Go to the Releases section of this repository.
2. Download `NutriQuest.jar`.

### Run

Windows:

```bash
java -jar NutriQuest.jar
```

Linux:

```bash
java -jar NutriQuest.jar
```

MacOS:

```bash
java -jar NutriQuest.jar
```

---

# Running from Source

### Clone the repository

```bash
git clone https://github.com/Saalim398/NutriQuest-.git
cd NutriQuest-
```

### Build the project

```bash
mvn clean install
```

### Run

```bash
mvn javafx:run
```
